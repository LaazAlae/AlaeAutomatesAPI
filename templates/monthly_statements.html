API Integration Guide

Statement Processing Overview

The Statement Processing API allows you to upload PDF statements and Excel DNM (Do Not Mail) lists for automated processing and categorization.

PDF Analysis

Extracts company information and statement details from PDF files

DNM Integration

Cross-references companies with your Excel DNM list

Manual Review

Prompts for manual decisions on ambiguous cases

ZIP Downloads

Provides organized results in downloadable ZIP format

API Endpoints

POST /api/statement-processor Create processing session
POST /api/statement-processor/{id}/upload Upload PDF and Excel files
POST /api/statement-processor/{id}/process Process uploaded files
GET /api/statement-processor/{id}/questions Get manual review questions
POST /api/statement-processor/{id}/answers Submit manual review answers
GET /api/statement-processor/{id}/download Download processed results

Code Examples

JavaScript Integration

// Create session
const response = await fetch('/api/statement-processor', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' }
});
const { session_id } = await response.json();

// Upload files
const formData = new FormData();
formData.append('pdf', pdfFile);
formData.append('excel', excelFile);

await fetch(`/api/statement-processor/${session_id}/upload`, {
    method: 'POST',
    body: formData
});

// Process files
await fetch(`/api/statement-processor/${session_id}/process`, {
    method: 'POST'
});
